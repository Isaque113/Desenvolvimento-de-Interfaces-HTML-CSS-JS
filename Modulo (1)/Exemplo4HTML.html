Estrutura de um Documento HTML

Um documento html é composto por várias tags estruturantes que são interpretadas pelo navegador, essas tags tem finalidades diferentes e são utilizadas para melhorar a performances, usabilidade dentre outras funções.

A seguir descrevemos as principais tags, utilizadas em um documento html.

Instrução DOCTYPE html

A instrução DOCTYPE html é utilizada pra indicar ao navegador a versão do html que será utilizada no documento. Ésssa întrução deve vir na primeira linha do documento html, sua definição é importante pois se não definirmos delgamos essa função ao navegador o que não é adequado.

Tag html

a tag html, é uma tag estruturante, essa tag é uma tag raiz que defini a estrutura geral do documento html. Outras tags estruturantes tais como head e body devem estar contindas nela.

Tag head

A tag head é utilizada para configurações do navegador e consequentemente do documento html, as ifnormações contidas

Tag body

A tag body é utilizada para exuibir o conteudo visivel para o usuário e nela temos outras tags, elementos html que deverão ser interpretados pelo navegador e mostradas coretamente para o usuário.
