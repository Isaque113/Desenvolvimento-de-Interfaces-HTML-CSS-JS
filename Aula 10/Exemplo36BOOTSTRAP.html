bootstrap

0 Bootstrap é framework front-end (html, css e javascript) muito popular e que foi criado pelo antigo Twiter. Esse framework possui diversos componentes que ajudam o desenvolvedor criar interfaces para seus sistemas.

Ele foi construido de forma que você possa usar as suas classes que já estão prontas, são usadas para estilizar documentos html.

Container

Um containere uma caixa ou recipiente que contém elementos html dentro. Seu objetivo maior éo de organizar esses elementos, controlando como esses elementos serão mostrados na tela.

Parea utiliazar um seletor de classe do bootstrap temos 2 opções: usar o seletor container ou o container-fluid.
