Exercicio 5 HTML - Principais Elementos HTML

Os principais elementos HTML incluem tags para estrutura, texto, mídia, e para todo tipo de interação, essas tags são interpretadas pelo navegador e utilizadas para a construção do documento HTML. A seguir descrevemos as principais tags não estruturantes.

Tag p - parágrafo

A tag parágrafo é representada pela letra p dentro do sinal de diamente e é utilizada para definir um parágrafo no documento html. Neste perágrafo estamos utilizando a tag para exibir o parágrafo para o usuário.
