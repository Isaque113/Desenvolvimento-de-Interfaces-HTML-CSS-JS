Exemplo 37 - Bootstrap - Cores Contextuais

O Bootstrap utiliza um sistema de cores contextuais para facilitar a marcação e a estilização de elementos com base em seu significado ou proposito. Essas classes de cores são usadas para dar significado visual aos elementos html. Geralmente utilizamos essas cores em botões, alertas, barras de navegação e outros elementos que são utilizados em interfaces gráficas.
